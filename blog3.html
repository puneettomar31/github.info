Deploying Laravel on Linux/FTP

Published on Dec 2024 | by Puneet Tomar

Deployment is an essential part of web development. I’ve deployed multiple Laravel projects on Linux servers using FTP and SSH workflows.

Steps I Follow:

  1. Set up Linux hosting environment (Apache/Nginx + PHP + MySQL)
  2. Upload Laravel files via FTP or Git
  3. Configure .env with database and mail settings
  4. Run php artisan migrate --seed for DB setup
  5. Set correct folder permissions for storage and bootstrap/cache
  6. Configure virtual hosts or .htaccess for public folder

Pro tip: Always test on a staging server before going live, and use version control for smooth updates.
